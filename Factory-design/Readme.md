1) Transportation Ticket Generator
Write a Java program demonstrating the Factory Design Pattern for a Transportation Ticket 
System.
Based on user input (BUS, TRAIN, FLIGHT), the system should generate different ticket objects:
• BusTicket
• TrainTicket
• FlightTicket
Your program must include:
• a Ticket interface
• concrete ticket classes
• a TicketFactory class
• a main method to test the system.



2) Animal Sound Generator
Write a Java program using the Factory Design Pattern to produce different animal sound objects.
Possible animal types:
• Dog
• Cat
• Cow
The program must include:
• an Animal interface
• concrete classes implementing sound()
• an AnimalFactory
• a main method to test based on user input.



3) Message Encryption Factory
Create a Java program demonstrating the Factory Pattern to generate different encryption 
strategies:
• AESEncryption
• DESEncryption
• Base64Encryption
Your solution must contain:
• an Encryption interface
• concrete encryption classes
• an EncryptionFactory
• a main method that asks the user which encryption type to use.



4) Food Ordering System
Write a Java program using the Factory Design Pattern to prepare different food items:
• Pizza
• Burger
• Pasta
The program should include:
• a Food interface
• concrete food classes implementing a prepare() method
• a FoodFactory class
• a main method where the user chooses a food item and the system prepares it.



5) Report Generator Factory
Write a Java program demonstrating the Factory Pattern to create different types of reports:
• PDFReport
• ExcelReport
• HTMLReport
Your program must include:
• a Report interface
• concrete classes
• a ReportFactory
• a main method that generates a report type based on user input.
